## آپ کیا سیکھیں گے

یہ ٹیکسٹ بک آپ کو ایک فزیکل ای انجینئر بننے کے قابل بنائے گی جو انٹیلیجنٹ روبوٹس تیار کر سکتا ہے۔ آپ نظریاتی سمجھ سے حقیقی درخیوں تک جائیں گے:

- **روبوٹ آپریٹنگ سسٹم (ROS 2)**: جدید روبوٹس کے نروس سسٹم کا ماسٹر کریں
- **ڈیجیٹل ٹوئن**: ورچوئل ماحول میں روبوٹس کی تیاری اور ٹیسٹنگ کریں
- **ای میں یکجتہ**: ایج کمپیوٹنگ ڈیوائسز پر ادراک اور نیویگیشن سسٹمز ڈپلائی کریں
- **وژن-لینگویج-ایکشن**: ایسے روبوٹس تیار کریں جو قدرتی زبان کے حکم کو سمجھ سکتے ہیں
- **ہارڈ ویئر ایکیجمنٹ**: حقیقی ہیومنوائڈ روبوٹس کے ساتھ کام کریں اور ہارڈ ویئر کی ضروریات کو سمجھیں

اس کورس کے اختتام پر، آپ کے پاس انٹیلیجنٹ ہیومنوائڈ روبوٹس کو تیار کرنے کے مہارتیں ہوں گی جو قدرتی زبان کے ذریعے فزیکل دنیا کو سمجھ سکتے اور اس کے ساتھ تعامل کر سکتے ہیں۔
